# 智能停车场管理系统 java788

## 项目概述

智能停车场管理系统是基于Vue、SpringBoot和MySQL的前后端分离源码实现，旨在为车主和停车场管理员提供便捷、高效的管理工具。

## 技术栈

- **后台框架：**
  - SpringBoot
  - MyBatis-Plus
- **UI界面：**
  - Vue
  - ElementUI
  - Axios
  - Node.js（前后端分离）
- **数据库：**
  - MySQL

## 功能模块

- 车主管理模块
- 车辆管理模块
- 停车场模块
- 停车记录模块
- IC卡模块
- IC卡挂失模块
- 用户管理
- 部门管理
- 角色管理
- 菜单管理
- 日志管理
- 数据字典管理
- 文件管理
- 图表展示

系统基于角色的访问控制，可将权限精确到按钮级别，支持自定义角色和权限分配。

## 系统角色

系统适用于需要设计精确权限约束的场景，主要服务于车主和停车场管理员。

## 运行环境

- 前端：兼容现代Web浏览器
- 后端：Java 1.8及以上版本环境

## 部署步骤

1. 配置数据库环境（MySQL）
2. 部署后端服务（SpringBoot应用）
3. 部署前端服务（Vue应用）
4. 启动前后端服务，进行系统配置
